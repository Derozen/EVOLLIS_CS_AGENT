# React + Vite

This is the My react app for the agent. The interface is very smplistic.
- [Link to the Live APP](https://evollis-cs-agent.vercel.app/) 

## Installation
You should have node Js installed on your computer as a prerequisite. In case of dependencies error, try using a node version <23
```
git clone https://github.com/Derozen/EVOLLIS_CS_AGENT.git
cd EVOLLIS_CS_AGENT
npm install
npm run dev

```
